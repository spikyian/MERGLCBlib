Detailed Description

This is the service descriptor and the main entry point to a service.

The module's application code should add the service descriptors to the services array for any of the services which it requires to be implemented. The merglcb library code then uses the pointers in the service descriptor to access the service's functionality. It is the responsibility of the writer of the service to provide a populated singleton for the service that can be used by the module's application code.
